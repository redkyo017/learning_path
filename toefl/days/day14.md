## Block 1 — British Shadowing (20 min)

**Source:** English with Lucy "vowel sounds British" — search "English with Lucy British vowel sounds" on YouTube

**Week 2 Phonology Focus: Sentence stress + weak forms**
Specifically drill:
- Stress shift for contrast: "I said BLUE, not GREEN" — stressed item is louder, longer, higher pitch
- Weak form "of": "a lot of" → /ə lɒt əv/, "because of" → /bɪkɒz əv/, "kind of" → /kaɪnd əv/
- Weak form "the" before consonants: "the result" → /ðə rɪzʌlt/ (not /ðiː/); "the" before vowels: "the outcome" → /ðɪ aʊtkʌm/

**Protocol:**
1. Listen once, no transcript (2 min)
2. Listen again with transcript (2 min)
3. Shadow at 0.75x speed on YouTube (5 min)
4. Shadow at full speed (5 min)
5. Record 30 seconds of yourself. Play back. Note: is "of" reduced in phrases like "kind of" and "a lot of"? Is "the" correctly /ðə/ before consonants and /ðɪ/ before vowels? (6 min)

## Block 2 — Speaking: Task 3 Academic Concept (30 min)

**Template:** templates/speaking_templates.md → Task 3

**Prompt:** "operant conditioning" — the academic reading defines the concept (behaviours are shaped by their consequences: reinforcement increases a behaviour, punishment decreases it); the lecture gives a concrete example of training a dog using treats and corrections (TST Prep style).

**Steps:**
1. Read the Task 3 template (2 min)
2. Prep: 30 seconds — note the definition from the reading + the professor's example + how the example shows both reinforcement and punishment
3. Record response (60 seconds) — use the template opening: "According to the reading, operant conditioning is… The professor illustrates this concept by…"
4. Play back. Score yourself:
   - Delivery (1-4): ___
   - Language (1-4): ___
   - Development (1-4): ___
5. Identify ONE fix (e.g., definition imprecise, example not clearly linked, needed to name both reinforcement and punishment)
6. Record again applying that fix

## Block 3 — Reading: Academic Skimming (25 min)

**Topic:** Biology

Search "TOEFL reading practice biology evolution passage" for a 700-word practice passage.

**Skimming protocol:**
1. Read title only → predict: what is this about? (30 sec)
2. Read ONLY the first sentence of each paragraph (2 min)
3. Write: thesis = ___, main arguments = (1) ___ (2) ___ (3) ___
4. Answer the questions WITHOUT re-reading (15 min)
5. Review wrong answers: A=vocab B=inference C=main idea D=detail E=rhetorical purpose (5 min)
