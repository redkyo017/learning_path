## Block 1 — British Shadowing (20 min)

**Source:** English with Lucy "British pronunciation" — search "English with Lucy British pronunciation" on YouTube

**Week 1 Phonology Focus: Non-rhotic vowels + final consonant clusters**
Specifically drill:
- Vowel contrast: British "bath" → /bɑːθ/ (long a) vs. "cat" → /kæt/ (short a) — the BATH-TRAP split
- Non-rhotic "ure": "sure" → /ʃɔː/, "pure" → /pjʊə/, "cure" → /kjʊə/ — no r, glide to schwa
- Final clusters: "acts" → /ækts/, "asks" → /ɑːsks/, "tasks" → /tɑːsks/ — practise each cluster slowly then at speed

**Protocol:**
1. Listen once, no transcript (2 min)
2. Listen again with transcript (2 min)
3. Shadow at 0.75x speed on YouTube (5 min)
4. Shadow at full speed (5 min)
5. Record 30 seconds of yourself. Play back. Note: is your "bath" long /bɑːθ/ (not the American short /bæθ/), and are your final clusters fully articulated? (6 min)

## Block 2 — Speaking: Task 3 Academic Concept (30 min)

**Template:** templates/speaking_templates.md → Task 3

**Prompt:** "cognitive dissonance" — the academic reading defines the concept; the lecture gives a concrete example of a person who continues smoking despite knowing the health risks (NoteFull Task 3 style).

**Steps:**
1. Read the Task 3 template (2 min)
2. Prep: 30 seconds — note the definition from the reading + the professor's example + how the example illustrates the concept
3. Record response (60 seconds) — use the template opening: "According to the reading, cognitive dissonance is… The professor illustrates this by…"
4. Play back. Score yourself:
   - Delivery (1-4): ___
   - Language (1-4): ___
   - Development (1-4): ___
5. Identify ONE fix (e.g., definition too vague, example not clearly linked to concept, pace)
6. Record again applying that fix

## Block 3 — Reading: Academic Skimming (25 min)

**Topic:** History/civilisation

Search "TOEFL reading practice history civilisation passage" for a 700-word practice passage.

**Skimming protocol:**
1. Read title only → predict: what is this about? (30 sec)
2. Read ONLY the first sentence of each paragraph (2 min)
3. Write: thesis = ___, main arguments = (1) ___ (2) ___ (3) ___
4. Answer the questions WITHOUT re-reading (15 min)
5. Review wrong answers: A=vocab B=inference C=main idea D=detail E=rhetorical purpose (5 min)
