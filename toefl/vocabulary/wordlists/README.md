## How to Study (Daily Integration)

### Instead of Magoosh (fully offline):
Pick 20 words from today's theme file. For each:
1. Read the definition — cover it — say it from memory
2. Read the example sentence — identify the collocation used
3. Write your own sentence using one of the listed collocations
4. Say your sentence aloud (British RP: stress content words)

Log your sentences in `../production_notebook.md`.

### Alongside Magoosh:
Use the theme files as a reference when you encounter an unfamiliar TOEFL word.
The thematic grouping helps — if you look up "eutrophication", you will see 20 related ecology words nearby and learn them in context.
