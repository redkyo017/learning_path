# Linear Algebra 30-Day Plan — Content Index

worked examples, exercises with complete solutions, and (on content days) a
code lab. Start with `docs/superpowers/plans/2026-07-08-linear-algebra-30-day-plan.md`
for the daily schedule and time budget; use this index to jump straight to a
given day's material.

Code labs: `starter_code/dayNN_*.py` has a TODO skeleton to fill in
yourself; `solutions/dayNN_*.py` is the checked, working reference —
open it only after attempting the skeleton.
